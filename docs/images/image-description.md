# This document describes the images in this folder.

- **start_screen.png**: This is the Home screen of the app. It shows upcoming workouts. In this example there are two defined workouts (`A` and `B`), so the list shows `Workout B`, `Workout A`, and then `Workout B` again. It has a floating button to start the next planned workout, which should send the user into the active workout flow.
- **workout_overview.png**: This shows the workout overview inside the `Programs` area, with the `Workouts` tab selected. It has a floating button to add a new workout. The six dots on the left are for reordering workouts. The three dots on the right open a context menu where a workout can be deleted. Pressing a workout opens **edit_workout.png**.
- **edit_workout.png**: This is the editor for a workout. The six dots on the left reorder exercises. The three dots on the right open a context menu to delete an exercise. It includes a floating button to add new exercises. Pressing an exercise opens **edit_exercise.png**.
- **edit_exercise.png**: This is the editor for an exercise, with the `Weight` tab selected. It shows sets x reps, the weight for the next workout, progression and deload settings, and plate guidance.
- **progress_exercise.png**: This shows a graph of the weight lifted for a given exercise over the last three months. This screen is reachable from **edit_exercise.png**.
- **history_exercise.png**: This shows the full history of all non-warmup sets done for a given exercise. This screen is reachable from **edit_exercise.png**.
- **history_workout.png**: This shows the history of completed workouts. This is one of the main items in the bottom navigation.
- **history_calendar.png**: This is **history_workout.png** in a calendar layout, marking dates where a workout was completed.
- **progress.png**: This screen shows progression graphs for all exercises defined in any workout. Pressing one of them takes the user to **progress_exercise.png** for that exercise.
- **workout_in_progress_0.png**, **workout_in_progress_1.png**, **workout_in_progress_2.png**: These show the workout screen at different stages, starting with no sets marked and then showing more completed sets. A set (gray circle) can be clicked to mark it as completed (red circle). Pressing it again lowers the reps accomplished. Each set also has a context menu where you can reset the set, log reps, change weight, or remove the set completely. After the last set there is a gray circle with a plus to quickly add a new set, as shown for Deadlift.
- **workout_warmup.png**: This shows the planned warmup sets for the current workout. None are marked yet, but it should work similarly to **workout_in_progress_0.png** through **workout_in_progress_2.png**. When all warmup sets for an exercise are complete, it should switch back to the `Workout` pane in the top menu.
